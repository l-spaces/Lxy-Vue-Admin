# result Token

Use these variables through `theme.components.Result`. This document lists definitions only and does not include values.

```vue
<a-config-provider
  :theme="{
    token: {
      // Global token configuration
    },
    components: {
      Result: {
        // Token name: value
      },
    },
  }"
>
  ...
</a-config-provider>
```

## Token List

| Token | Type | Description |
| --- | --- | --- |
| `titleFontSize` | `number` | Title font size |
| `subtitleFontSize` | `number` | Subtitle font size |
| `iconFontSize` | `number` | Icon size |
| `extraMargin` | `CSSProperties` | Margin of extra area |
