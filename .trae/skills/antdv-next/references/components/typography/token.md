# typography Token

Use these variables through `theme.components.Typography`. This document lists definitions only and does not include values.

```vue
<a-config-provider
  :theme="{
    token: {
      // Global token configuration
    },
    components: {
      Typography: {
        // Token name: value
      },
    },
  }"
>
  ...
</a-config-provider>
```

## Token List

| Token | Type | Description |
| --- | --- | --- |
| `titleMarginTop` | `string \| number` | Margin top of title |
| `titleMarginBottom` | `string \| number` | Margin bottom of title |
