# anchor Token

Use these variables through `theme.components.Anchor`. This document lists definitions only and does not include values.

```vue
<a-config-provider
  :theme="{
    token: {
      // Global token configuration
    },
    components: {
      Anchor: {
        // Token name: value
      },
    },
  }"
>
  ...
</a-config-provider>
```

## Token List

| Token | Type | Description |
| --- | --- | --- |
| `linkPaddingBlock` | `number` | Vertical padding of link |
| `linkPaddingInlineStart` | `number` | Horizontal padding of link |
