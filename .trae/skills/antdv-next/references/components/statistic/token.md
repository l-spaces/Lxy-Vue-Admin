# statistic Token

Use these variables through `theme.components.Statistic`. This document lists definitions only and does not include values.

```vue
<a-config-provider
  :theme="{
    token: {
      // Global token configuration
    },
    components: {
      Statistic: {
        // Token name: value
      },
    },
  }"
>
  ...
</a-config-provider>
```

## Token List

| Token | Type | Description |
| --- | --- | --- |
| `titleFontSize` | `number` | Title font size |
| `contentFontSize` | `number` | Content font size |
