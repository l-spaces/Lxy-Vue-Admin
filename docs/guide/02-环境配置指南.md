## 2. 安装步骤

### 2.1 安装 Node.js

**方式一：使用 nvm（推荐）**

```bash
# 安装 nvm
# Windows: https://github.com/coreybutler/nvm-windows/releases
# macOS/Linux: https://github.com/nvm-sh/nvm

# 安装 Node.js
nvm install 22.18.0
nvm use 22.18.0

# 验证安装
node -v
npm -v
```

**方式二：官网下载**

访问 [Node.js 官网](https://nodejs.org/) 下载对应版本安装包。

### 2.2 安装 pnpm

```bash
# 使用 npm 安装 pnpm
npm install -g pnpm

# 验证安装
pnpm -v
```

### 2.3 克隆项目

```bash
# 克隆项目
git clone https://gitee.com/my_spaces/lxy-vue-admin.git

# 进入项目目录
cd lxy-vue-admin
```

### 2.4 安装依赖

```bash
# 安装项目依赖
pnpm install

# 或使用 bootstrap 命令
pnpm bootstrap
```

### 2.5 启动开发服务器

```bash
# 启动 web-antd 应用
pnpm dev:antd

# 或使用通用开发命令
pnpm dev
```

启动成功后，访问：`http://localhost:5666`

### 3.3 代理配置

开发环境下需要配置代理以解决跨域问题。

**配置文件位置**：`apps/web-antd/vite.config.ts`

```typescript
export default defineApplicationConfig({
  server: {
    host: true,
    port: VITE_PORT,
    proxy: {
      '/prod-api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/prod-api/, ''),
        ws: true,
      },
    },
  },
});
```

**代理配置说明**：

| 配置项 | 说明 |
|--------|------|
| `target` | 后端服务地址 |
| `changeOrigin` | 改变请求源 |
| `rewrite` | 重写路径 |
| `ws` | 支持 WebSocket 代理 |

## 4. 生产环境打包

### 4.1 构建命令

```bash
# 生产环境打包
pnpm build:antd

# 测试环境打包
pnpm build:antd:test

# 构建分析
pnpm build:analyze

# 构建所有应用
pnpm build
```

### 4.2 构建输出

打包完成后，产物输出到 `apps/web-antd/dist` 目录：

```
dist/
├── index.html           # 入口 HTML 文件
├── assets/              # 静态资源目录
│   ├── index-xxx.js     # JS 文件
│   ├── index-xxx.css    # CSS 文件
│   └── ...              # 其他资源
├── favicon.ico          # 网站图标
└── dist.zip             # 打包压缩文件（可选）
```

### 4.3 构建配置

**生产环境配置**：`.env.production`

```properties
VITE_BASE=/vben

# 是否开启压缩
VITE_COMPRESS=gzip

# 是否开启 PWA
VITE_PWA=false

# vue-router 的模式
VITE_ROUTER_HISTORY=history

# 是否注入全局 loading
VITE_INJECT_APP_LOADING=true

# 打包后是否生成 dist.zip
VITE_ARCHIVER=true

# 后端接口地址
VITE_GLOB_API_URL=/prod-api
```

## 5. 部署配置

### 5.1 Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/dist;
    index index.html;

    # 开启 gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;
    gzip_min_length 1000;

    # 静态资源缓存
    location /assets {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /prod-api/ {
        proxy_pass http://backend-server:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### 5.2 Docker 部署

**Dockerfile 示例**：

```dockerfile
# 构建阶段
FROM node:22-alpine AS builder

WORKDIR /app

# 安装 pnpm
RUN npm install -g pnpm

# 复制 package.json
COPY package.json pnpm-lock.yaml pnpm-workspace.yaml ./
COPY apps/web-antd/package.json ./apps/web-antd/

# 安装依赖
RUN pnpm install --frozen-lockfile

# 复制源代码
COPY . .

# 构建
RUN pnpm build:antd

# 生产阶段
FROM nginx:alpine

# 复制构建产物
COPY --from=builder /app/apps/web-antd/dist /usr/share/nginx/html

# 复制 nginx 配置
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

**构建和运行**：

```bash
# 构建镜像
docker build -t lxy-vue-admin .

# 运行容器
docker run -d -p 80:80 lxy-vue-admin
```

## 7. 常见问题

### 7.1 Node 版本不兼容

**问题**：提示 Node 版本过低

**解决方案**：

```bash
# 使用 nvm 切换 Node 版本
nvm install 22.18.0
nvm use 22.18.0

# 验证版本
node -v
```

### 7.2 pnpm 依赖安装失败

**问题**：`pnpm install` 报错

**解决方案**：

```bash
# 清理缓存
pnpm store prune

# 删除 node_modules 和 lock 文件
rm -rf node_modules pnpm-lock.yaml

# 重新安装
pnpm install
```

### 7.3 热更新不生效

**问题**：修改代码后页面不刷新

**解决方案**：

1. 检查 Vite 配置中的 `server.hmr` 配置
2. 清除浏览器缓存
3. 重启开发服务器

### 7.4 跨域问题

**问题**：开发环境请求后端报跨域错误

**解决方案**：

在 `apps/web-antd/vite.config.ts` 中配置代理：

```typescript
server: {
  proxy: {
    '/prod-api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/prod-api/, ''),
    },
  },
}
```
