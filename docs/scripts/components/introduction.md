# 介绍

::: info README

本节记录了框架组件，包括它们的使用模式、配置点和主要 API。如果内置封装不能满足您的需求，您可以直接使用原生组件或构建自己的抽象。

:::

## 布局组件

布局组件通常用作页面内容区域内的顶级容器。它们提供共享的布局样式和一些基础行为。

## 通用组件

通用组件包括常用的 UI 构建块，如模态框、抽屉、表单和基于 API 的选择器。它们大多数基于共享的 Tailwind CSS 和 shadcn-vue 原语实现，同时仍允许每个应用将其适配到自己的 UI 库。
