# 常见问题解决方案

## 简介

以下问题来自当前仓库源码注释、实现逻辑与已存在行为，不包含推测性结论。

## 适用范围

- 登录与权限异常
- 请求与加密异常
- 字典与表单展示异常

## 对应源码目录或关键文件

- `apps/web-antd/src/utils/http/index.ts`
- `apps/web-antd/src/utils/http/helper.ts`
- `apps/web-antd/src/store/dict.ts`
- `apps/web-antd/src/preferences.ts`
- `apps/web-antd/src/adapter/component/index.ts`

## 核心机制或功能说明

### 1. SM2/RSA 加密问题

- 当前实现默认 `RsaEncryption + AesEncryption`。
- 源码注释已明确：前端 `sm-crypto` 场景下公钥需 `04` 开头，否则可能失败。
- `axios` 对字符串包引号的问题已通过 `transformRequest` 处理。

### 2. 微服务网关 401 兼容问题

- 源码注释指出：网关若返回 `text/plain`，JSON 分支逻辑不会生效。
- 当前策略偏向后端网关修正，而不是在前端增加复杂兼容分支。

### 3. 字典重复请求

- `useDictStore` 使用 `dictRequestCache` 对同名字典请求加锁，避免页面多处重复并发。

### 4. KeepAlive + 语言切换下 placeholder 不更新

- 组件适配层使用计算属性生成 placeholder，避免缓存场景展示不刷新。

### 5. 数组参数和区间参数传递异常

- GET/DELETE 默认启用 `paramsSerializer`，数组采用 `repeat` 格式，解决后端参数接收问题。

### 6. 登录过期弹窗模式限制

- `preferences.ts` 注释明确：当前不启用 modal 形式的登录过期处理，避免菜单与页面状态不一致。

### 7. ClientID 绑定异常

- HTTP 封装注释已记录：`ClientID` 头在部分后端场景可能被错误绑定到实体类。

## 使用方式、扩展方式或注意事项

- 排障优先查看浏览器 Network 与 `utils/http` 中实际 header/body。
- 请求异常时先确认环境变量和代理，再检查 token/语言头/加密开关。
- 若需扩展故障处理策略，优先在 `utils/http/*` 做集中改造。
