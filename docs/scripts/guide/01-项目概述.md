## 1. 项目简介

**Lxy-Vue-Admin** 是基于 [Vben Admin 5](https://github.com/vbenjs/vue-vben-admin) 和 [Ant Design Vue Next](https://github.com/antdv-next/antdv-next) 开发的企业级中后台管理系统前端项目。该项目与 **Lxy-Spring** 后端框架深度集成，提供了一套完整的前端解决方案。

### 项目定位

- 企业级中后台管理系统前端框架
- 基于 Monorepo 架构的现代化前端项目
- 与 Lxy-Spring 后端深度集成的前端实现

### 版本信息

| 组件 | 版本 |
|------|------|
| Vben Admin | 5.7.0 |
| Vue | 3.5.13 |
| Ant Design Vue Next | 4.2.6 |
| TypeScript | 5.x |
| Vite | 5.x |

## 3. 项目特性

### 核心特性

- **Monorepo 架构**：使用 pnpm workspace 和 Turbo 进行多包管理，代码复用性高
- **TypeScript 支持**：完整的类型定义和类型检查，代码更加健壮
- **组件化开发**：基于 Ant Design Vue Next 的高度封装组件系统
- **动态路由**：支持后端动态路由配置和权限控制
- **多租户支持**：内置多租户管理和切换功能
- **数据加密**：支持 RSA/AES 前后端数据加密传输，支持国密 SM2/SM4
- **国际化**：完整的多语言支持（中文/英文），可扩展更多语言
- **响应式设计**：适配多种屏幕尺寸和设备
- **主题定制**：支持主题切换和个性化配置
- **代码规范**：完善的代码规范和自动化检查工具

### 功能特性

- **认证授权**：支持账号密码登录、验证码登录、二维码登录、第三方登录
- **权限管理**：基于角色和权限码的双重验证机制
- **菜单管理**：动态菜单生成，支持外链、内嵌 iframe
- **字典管理**：系统字典数据管理，支持缓存
- **文件管理**：支持多种 OSS 存储（本地、MinIO、阿里云、腾讯云、七牛云）
- **工作流**：集成 Flowable 工作流引擎
- **代码生成**：支持前后端代码生成
- **系统监控**：在线用户、操作日志、登录日志、缓存监控

## 6. 快速开始

### 环境要求

- **Node.js**: ^20.19.0 或 ^22.18.0 或 ^24.0.0
- **pnpm**: >= 10.0.0
- **Git**: 最新版本

### 安装步骤

```bash
# 1. 克隆项目
git clone https://gitee.com/my_spaces/lxy-vue-admin.git
cd lxy-vue-admin

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm dev:antd

# 4. 访问项目
# 浏览器打开 http://localhost:5666
```

### 默认登录信息

- **用户名**: `admin`
- **密码**: `123456`
