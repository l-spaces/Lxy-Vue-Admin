# 环境配置指南

## 简介

本文档说明本项目的本地开发环境要求、环境变量来源以及开发代理配置。

## 适用范围

- 首次搭建开发环境
- 联调前环境变量核对
- 构建前配置检查

## 对应源码目录或关键文件

- `package.json`
- `apps/web-antd/.env`
- `apps/web-antd/.env.development`
- `apps/web-antd/.env.production`
- `apps/web-antd/.env.test`
- `apps/web-antd/vite.config.ts`
- `packages/effects/hooks/src/use-app-config.ts`

## 核心机制或功能说明

### 环境要求

- Node.js：`^20.19.0 || ^22.18.0 || ^24.0.0`
- pnpm：`>=10.0.0`

### 常用命令

```bash
pnpm install
pnpm dev:antd
pnpm build:antd
pnpm docs:dev
```

### 环境变量读取机制

- 开发环境：直接读取 `import.meta.env`
- 生产环境：读取 `window._VBEN_ADMIN_PRO_APP_CONF_`
- 解析入口：`packages/effects/hooks/src/use-app-config.ts`

### 关键环境变量（当前实现）

- `VITE_GLOB_API_URL`：后端基地址
- `VITE_GLOB_APP_CLIENT_ID`：请求头 `ClientID`
- `VITE_GLOB_ENABLE_ENCRYPT`：请求加密总开关
- `VITE_GLOB_RSA_PUBLIC_KEY` / `VITE_GLOB_RSA_PRIVATE_KEY`：RSA 密钥
- `VITE_GLOB_SSE_ENABLE` / `VITE_GLOB_WEBSOCKET_ENABLE`：消息通道开关
- `VITE_ROUTER_HISTORY`：`history/hash`

### 开发代理（当前 `apps/web-antd/vite.config.ts`）

- 代理前缀：`/api`
- 目标地址：`http://localhost:8080`
- 路径重写：`^/api -> ''`
- `ws: true`

## 使用方式、扩展方式或注意事项

- `.env.development` 中 `VITE_GLOB_API_URL` 和本地 `vite proxy` 可以并存；联调时需确认请求实际走向。
- 开启加密前需保证后端密钥和解密链路一致。
- 修改 `preferences` 或关键环境变量后，建议清理浏览器 `localStorage` 再验证。
