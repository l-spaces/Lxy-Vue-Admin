# 常用命令

## 简介

本页汇总当前仓库 `package.json` 中可直接执行的高频命令。

## 适用范围

- 日常开发
- 代码检查
- 构建与预览

## 对应源码目录或关键文件

- `package.json`
- `apps/web-antd/package.json`
- `docs/package.json`

## 核心机制或功能说明

### 根目录高频命令

```bash
pnpm install
pnpm dev
pnpm dev:antd
pnpm docs:dev
pnpm lint
pnpm check:type
pnpm check
pnpm build
pnpm build:antd
pnpm build:antd:test
pnpm docs:build
```

### `apps/web-antd` 子包命令

```bash
pnpm -F @vben/web-antd run dev
pnpm -F @vben/web-antd run build:prod
pnpm -F @vben/web-antd run build:test
pnpm -F @vben/web-antd run build:analyze
pnpm -F @vben/web-antd run typecheck
```

### `docs` 子包命令

```bash
pnpm -F @lxy/docs run dev
pnpm -F @lxy/docs run build
pnpm -F @lxy/docs run preview
```

## 使用方式、扩展方式或注意事项

- 优先使用根命令，避免跨子包执行导致参数不一致。
- `build:analyze` 仅分析主应用包体，不等同于文档站点分析。
- 执行 `check` 前建议先 `pnpm install` 保证 workspace 依赖完整。
