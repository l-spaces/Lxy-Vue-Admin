# API 接口文档

## 简介

本页保留原文件名作为 API 分类总入口，避免历史链接失效。具体内容已按“请求层、模块组织、核心接口”拆分到独立文档。

## 适用范围

- 通过旧链接进入 API 文档的场景
- 需要快速跳转到接口细分文档的场景

## 对应源码目录或关键文件

- `apps/web-antd/src/api`
- `apps/web-antd/src/utils/http`

## 阅读入口

- API 分类首页：[README](./README.md)
- 请求封装与拦截器：[请求层封装](./请求层封装.md)

- 认证/菜单/用户等接口说明：[核心接口说明](./核心接口说明.md)

## 核心机制或功能说明

- `apps/web-antd/src/api/index.ts` 当前仅 `export * from './core'`，表示 `#/api` 只覆盖核心接口。
- 系统、监控、工作流、工具模块通过 `#/api/<domain>/<module>/...` 直接导入。
- 请求统一走 `alovaInstance`，具体处理链路见 [请求层封装](./请求层封装.md)。

## 使用方式、扩展方式或注意事项

- 新增 API 文档时，优先补到对应专题页，不再把全部接口堆在本页。
- 若需要保留对外固定链接，继续复用本文件作为目录跳转页。
